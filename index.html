CitySDK

The CitySDK is a javascript library that sits on top of the Census APIs, with a focus on American Community Survey data (ACS). It makes it easier to use the most commonly requested data from the ACS, along with geographic shape files in the GeoJSON format. Since it's also an open-source and community-led project, there are plug-ins that connect to other federal agencies like USDA, DOE, and HUD, and industry plug-ins that connect to city open data portals like ESRI and Socrata. If you are building a web app that requires Census data to be used with mapping tools like Google Maps or Leaflet, it's suggested you start with CitySDK in order to save time and code your app more quickly. If you need any additional APIs from Census that aren't included in the CitySDK, don't care about mapping, or don't what to use Javascript -- than you can simply use the other Census APIs directly without the CitySDK.

American Community Survey 5 Year

The American Community Survey (ACS) is an ongoing survey that provides data every year -- giving communities the current information they need to plan investments and services. The ACS covers a broad range of topics about social, economic, demographic, and housing characteristics of the U.S. population. Much of the ACS data provided on the Census Bureau's Web site are available separately by age group, race, Hispanic origin, and sex. The 5-year estimates from the ACS are "period" estimates that represent data collected over a period of time. The primary advantage of using multiyear estimates is the increased statistical reliability of the data for less populated areas and small population subgroups.
